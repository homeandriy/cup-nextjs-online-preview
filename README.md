# CUP Next.js Online Preview

An online cup preview and customization application built with Next.js.

## Stack

- Next.js 16 and React 19
- TypeScript
- Tailwind CSS
- Konva and React Three Fiber
- Cypress

## Getting started

1. Copy .env.example to .env.local and configure any required values.
2. Start the development environment with docker compose up --build.
3. Open http://localhost:3010.

## Useful commands

- docker compose up --build
- docker compose run --rm cypress
